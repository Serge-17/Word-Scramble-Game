# Word Scramble Game

![SwiftUI](https://img.shields.io/badge/SwiftUI-Framework-blue)

## Описание
**Word Scramble Game** - это классическая игра на основе слов, где игрок должен составлять слова из предоставленных букв и проверять их на правильность. Эта игра поможет вам улучшить ваши навыки словарного запаса и мышления.

## Особенности
- **Начало игры**: Случайный выбор корневого слова из списка слов.
- **Добавление слов**: Игроки могут вводить слова, которые проверяются на уникальность, возможность составления из букв корневого слова и наличие в словаре.
- **Статистика игрока**: Отображает количество угаданных слов и общее количество букв в угаданных словах.
- **Обработка ошибок**: Всплывающие окна предупреждают игрока о неверных действиях.
- **Рестарт игры**: Кнопка для перезапуска игры с новым корневым словом.
